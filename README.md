## Project Overview

AI Emergency Companion is a web application designed to provide assistance during emergencies using AI-powered features. The project includes interactive user interfaces and real-time support functionalities.

---

## Features

- Real-time AI assistance
- Responsive design for mobile and desktop
- Easy deployment with Vercel
- User-friendly interface

## Tech Stack

- **Frontend:** React / Next.js
- **Styling:** CSS / Tailwind CSS (if used)
